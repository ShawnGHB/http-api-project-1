Get Books

Query Parameters:

  • author(optional) - author's name to filter by
  • title(optional) - book title's name to filter by
  • year(optional) - year's name to filter by

Returns:

An Object using Book titles as key values, and to objects holding book data. Format: {title: {object}}

Examples:

Request: /getBook

Result: ["Title": {title, author, yr, genre[]}]

Request: /getBook?title4

Result: ["Titl4": {title4, author, yr, genre[]}]


Get Titles

Query Parameters:

  • title(optional) - book title's name to filter by

Returns:

An array of titles, case sensitive. Format: [string]

Examples:

Request: /gettitles

Result: ["Title", "Title", ...]

Request: /getTitles?titl

Result: ["titl", "Blah 'tit1' Blah"...]

Get Authors

Query Parameters:

  • author(optional) - author's name to filter by

Returns:

An array of authors, case sensitive. Format: [string]

Examples:

Request: /getAuthors

Result: ["Author", "Author", ...]

Request: /Author?autl

Result: ["autl" Bronte, "aut1" Dickens, ...]

Show Books

Query Parameters:

  • none

Returns:

A list of book objects. Format: [{}, {}, {}]

Examples:

Request: /getBookData

Result: [{"book1"}, {"book2"}, ...]

Add Book

Body Parameters(required):

  • author - Author who wrote the book
  • title - Name of the book
  • year - Year book was published

Returns:

Nothing unless an error, can check on the book with above functions

Examples:

Request: /addBook

Body: {author: "author1", title: 'Title1', year: ####}

Result: Created Successfully

Give Genre

Body Parameters(required):

  • title - Name of the book
  • genre - Comma separated list of genres that the book falls under

Returns:

Nothing unless an error, can check on the book with Get Books or Show Books

Examples:

Request: /addGenres

Body: { title: 'Title1', genres: 'genre1,genre2'}
